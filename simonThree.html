Simon Says

Try to copy the computer's moves by pressing the buttons above the screen that coorespond to the sphere's color. If you see the spheres explode and the screen turns red, you've guessed incorrectly. In strict mode, this will reset the game. In lenient mode, Simon will show you the pattern again, and give you another chance. You can adjust the display window with the mouse if you want a different view, just make sure you can see all of the spheres! You win the game if you complete level 20, but you can keep playing after that if you like. Good Luck!
